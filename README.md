# 📚 Library Management System (Python)

## 📌 Description

This is a simple and beginner-friendly **Library Management System** built using **Python**. The project is completely menu-driven and runs in the terminal/console.

It helps manage books in a library by allowing users to:

* Add new books
* View available books
* Issue books with student name, issue date, and duration
* Return books
* Calculate fine for late returns (₹10 per day)

This project is perfect for Python beginners who want to understand:

* Functions
* Modules
* Lists
* Dictionaries
* Conditional Statements
* Loops
* Date Handling in Python

---

## 🚀 Features

### ✅ Add Book

Add new books to the library collection.

### 📖 Show Books

Display all books currently available in the library.

### 📕 Issue Book

Issue a book with student name, issue date, and return duration.

### 📗 Return Book

Return the issued book back to the library.

### 💰 Fine System

Late returns are charged **₹10 per day** automatically.

### 📚 Dictionary Data Storage

Books data is managed using Python dictionaries.

## 📌 How It Works

When you run the project, you will see this menu:

1. Add Book
2. Show Books
3. Issue Book
4. Return Book
5. Exit

Choose any option by entering the number.

### Step 1:

### Step 3:

Open project folder.

### Step 4:

Run the main file.
